# MyBalanceCar

基于经典的STM32F103C8T6、MPU6050、MX1508、ESP8266、OLED等的平衡小车源代码

基于CubeMX的HAL库

新版本：2021.11.05

添加了双路直流有刷mos驱动电路，修改USB封装为Type-C，优化走线，兼容OLED模块或裸屏
